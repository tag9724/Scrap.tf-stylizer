Introduction

All of your CSS code will be injected in the head tag on all scrapTF pages ( exept : chat / forum ).

So your own custom CSS can be injected before or after the default style.css used by scrap.tf, so I recommend you be careful with your selections and don't forget to use !important with your rules.

All settings on your template can be used with some CSS variables ( see the next part ).
If you want to take a look at the template.css injected with your template for applying it follow this link
